# Evidence Notes

This folder collects reviewer-facing evidence that explains what PaperSwarm can verify today, and where the current demo still has explicit boundaries.

## Files

- `non-consensus-evidence.md` documents the medical RCT case where independent reviewer roles found defects that a single generic reviewer would likely miss.
- `schema-gap-report.md` documents which audit fields can be recovered from the cached medical run and which fields must be produced natively by future STEP implementations.

## Public-Repo Boundary

These notes are curated from the original hackathon workspace. Internal planning files, registration/travel materials, OAuth scripts, raw live EvoMap payloads, and unredacted local run state are intentionally excluded from this public repository.
