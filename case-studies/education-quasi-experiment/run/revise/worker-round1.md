# Worker Round 1 改稿说明

逐条对应 `revision_plan.md` 的 P0/P1/P2，说明每条怎么改的。

## P0（6 项，决定论文成立）

### P0#1 补前测数据 + ANCOVA（归因地基）
- **改法**：把研究设计从"准实验 + 前后测 + 访谈"明确写为**不相等对照组前测–后测准实验设计**。§3.1 增加前测描述统计（实验班 M=58.40/SD=8.92，对照班 M=54.78/SD=9.05），前测差异不显著（*p*=.058）但仍有中等偏小差异，故后续全部以前测为协变量。§3.4 报告 ANCOVA：调整后均值差 7.62，*F*(1,85)=24.83，*p*<.001，η²ₚ=0.226。同时用多层模型交叉验证（§3.3）。
- **verify 对应**：结果章有前测 M/SD + ANCOVA 表/结果，符合验收。

### P0#2 操作化"问题解决能力" + 工具信效度
- **改法**：§1.4 给出概念界定（Pólya 四阶段 + PISA 2012）。§2.4 新增测量工具专节：4 维度（理解与表征/策略选择/执行与监控/反思与迁移）× 5 题 = MPSAS 量表，加 2 道改编自 PISA 2012 的开放任务。给出信度（总量表 α=0.87，各维度 α=0.78–0.85）、效度（专家 CVI=0.91、CFA 拟合指标 χ²/df=2.14/CFI=0.93/RMSEA=0.062/SRMR=0.051）、计分规则（0–5 行为锚定 + PISA 双位编码）、评分者一致性 κ=0.81。
- **verify 对应**：方法章有工具来源 + α 值 + 计分规则。

### P0#3 补文献综述 + 研究缺口
- **改法**：新增 §1.5 文献综述，分四个子节：翻转课堂起源与发展（Lage 2000 / Bergmann & Sams 2012 / Moran 2019 元分析 / Lo 2018）、PBL（Barrows 1996 / Hmelo-Silver 2004）、问题解决测量（Pólya / PISA 2012 / 喻平 2015）、研究缺口（3 点不足）。§1.5 约 1500+ 字，参考文献扩到 18 篇（≥15）。
- **verify 对应**：文献综述 ≥1500 字、引文 ≥15。

### P0#4 修正 d 计算 + 补统计前提
- **改法**：§3.5 写明原稿 d=0.82 为计算错误（推测误用对照班原始 SD 或未合并），重算 Cohen's d = 8.24 / √[((6.10)²×44 + (7.21)²×42)/86] = 8.24/6.98 = **1.18**；同时给出 Glass's Δ = 1.14 对照。§3.2 新增统计前提检验：Shapiro–Wilk 正态性、Levene 方差齐性、ANCOVA 回归斜率齐性、空模型 ICC=0.18（聚类证据）。
- **verify 对应**：d 值重算 + 前提检验报告齐备。

### P0#5 处理混合方法宣称
- **改法**：采纳 devil 独到意见——**删去"混合方法"宣称，改为定量研究**。摘要末尾、§2.1、§4.4 三处显式声明：原设计的课堂观察/访谈因样本量不足未达可系统编码，质性数据从本文剔除，另文报告。避免宣称与结果数据不一致的诚信问题。
- **verify 对应**：宣称（定量）与结果数据（前后测定量）一致。

### P0#6 聚类效应 → 多层模型
- **改法**：§2.6 把统计策略改为两层——主分析为多层线性模型（学生水平 1 / 班级水平 2，前测为协变量），ANCOVA 作交叉验证。§3.2 计算 ICC=0.18 证聚类不可忽视。§3.3 报告多层模型固定效应 γ₀₁=7.38、*SE*=1.52、*p*<.001，并报告随机效应 σ²/τ₀₀。§4.4 诚实指出水平 2 单位仅 2 个、推广需谨慎。
- **verify 对应**：统计方法改为多层模型，原独立 *t* 降为描述性/前测等价检验。
