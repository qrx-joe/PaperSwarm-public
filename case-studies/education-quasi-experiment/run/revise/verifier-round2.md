## 关键算术独立复核（先验后判）

**复核 d 值（Round 1 命门项）**：
- 论文公式：合并方差 = [(6.10)²×44 + (7.21)²×42] / 86
- (6.10)² = 37.21；37.21 × 44 = **1637.24**
- (7.21)² = 51.9841；51.9841 × 42 = **2183.3322**
- 分子合计 = 1637.24 + 2183.3322 = **3820.5722**
- 合并方差 = 3820.5722 / 86 = **44.4253**
- 合并 SD = √44.4253 = **6.6654 ≈ 6.66**
- d = (82.31 − 74.07) / 6.66 = 8.24 / 6.66 = **1.2372 ≈ 1.24** ✓

论文 Round 2 写的合并 SD = 6.66、d = 1.24，**算术完全正确**。Round 1 打回成立、Round 2 改对了。

**前测等价性 t 值旁检**：均值差 58.40−54.78=3.62，合并 SD≈8.98，SE≈8.98×√(1/45+1/43)≈8.98×0.2156≈1.936，t≈3.62/1.936≈1.87。论文写 t(86)=1.92，略偏高但在合理区间（独立样本 t 在"合并 SD"vs"各自 SD"公式变体下有差异），**不算硬伤**。
