## 结构判读（供下游评审组队 + 各角色 prompt）

- **学科：临床医学 RCT** → 领域专家（domain）应按 **CONSORT 报告规范** + ICMJE 试验注册要求审查适应症定位、GLP-1 类药物证据对比
- **方法：定量 RCT**，含 ITT 原则 + **LOCF 填补**缺失数据 + ANCOVA 模型 + **8 个次要终点**比较 + 子组分析 → 方法论（method）应查随机化执行、盲法维持、脱落处理；**统计审稿人（statistician）需核验**：LOCF 偏倚、缺失模式假设、多重比较校正、效应量与 CI
- **涉人体研究**（240 例患者、各中心伦理审批、知情同意）→ **伦理审稿人（ethicist）需核验**：IRB 审批文号、知情同意流程、数据保护、利益冲突（制药公司资助？）、CONSORT 流程图透明度
- **脱落率不对称**（LX-204 18.3% vs 安慰剂 30.0%）+ LOCF → 统计与方法重点关注项
- **无代码/数据集/模型权重开放需求**（纯临床统计）→ 无需复现审稿人（reproducibility）
