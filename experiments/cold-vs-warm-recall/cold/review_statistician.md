# 生物统计审稿意见 — 利拉鲁肽 RCT
> 角色：statistician（独立评审，未见其他评审）

## 评分（1-5）
| 维度 | 分数 | 一句依据 |
|---|---|---|
| 主要分析的统计自洽性 | 1 | CI 与 p 值在数学上不可能同时成立，且 CI 不以点估计为中心 |
| 样本量与功效 | 1 | 完全没有 power 计算，N=200 无任何依据 |
| 缺失数据处理与敏感性分析 | 2 | LOCF + 2 倍不对称脱落，零敏感性分析 |
| 多重比较与选择性报告 | 2 | 次要终点名义 p<0.05、子组无交互 p、摘要挑樱桃 |

## 发现的缺陷清单（按严重度降序，每条必须编号）

1. [致命] 主要终点的 95% CI 与 p 值在数学上不可能同时成立，且 CI 不以点估计为中心
   - **引用**：摘要 / §3.2「体重组间差值 5.6%（95% CI 5.1–7.7%，p=0.02）」
   - **简析**：两层内部矛盾，任一层单独都足以推翻结果。(a) 95% CI 必然以点估计为中心对称，但该 CI 中点 (5.1+7.7)/2 = 6.4，不等于点估计 5.6，差 0.8 个百分点——对称估计量不可能产生这样的区间。(b) 即便强行按 CI 反推：SE = (7.7−5.1)/(2×1.96) = 0.663，z = 5.6/0.663 = 8.45，对应 p ≈ 3×10⁻¹⁷，绝非 0.02。反向由 p=0.02（z≈2.326）反推 CI，应为 [0.88, 10.32]，与报告的 [5.1, 7.7] 完全不符。可疑点：CI 中点 6.4 恰好等于利拉鲁肽组的组内减重幅度，强烈提示 CI 数字是组内值误植或编造。
   - **正确做法**：提交原始数据与 ANCOVA 输出，重新计算 SE 与 95% CI，确保区间对称、与 p 值一致。在查清前主要终点结论不可采信。

2. [重大] 完全缺失样本量与功效计算
   - **引用**：§2.4 统计分析仅写「主要终点采用 ANCOVA，以基线体重为协变量」，全文无 α、β、预设效应量、脱落补偿、样本量推导
   - **简析**：N=200 凭空给出，没有 power 计算，无法判断试验是否 adequately powered；也无法事后判断 p=0.02 的「显著」是真信号还是低功效下的夸张估计（low power → effect size inflation, type M error）。CONSORT 与 ICH-E9 均要求预设样本量理由。
   - **正确做法**：补 a priori 功效分析：明确 α（通常 0.05 双侧）、power（≥80%）、基于既往利拉鲁肽试验（如 SCALE）的预期组间差值与 SD、脱落补偿系数，反推 N 并与实际入组对照。

3. [重大] LOCF 填补 + 2 倍不对称脱落，且无任何敏感性分析
   - **引用**：§2.4「缺失数据以 LOCF 填补」；§3.1「利拉鲁肽组脱落 12.0%，安慰剂组脱落 24.0%」
   - **简析**：脱落率组间 2 倍差异（12% vs 24%）本身已提示缺失非随机（differential dropout），可能与疗效或不良事件相关——这构成 MNAR 嫌疑。LOCF 在 MNAR 下会产生有偏的治疗效应估计和偏小的 SE；且对持续下降的体重轨迹，LOCF 系统性低估真实脱落者的终值差异，方向上对获益方有利。论文未报告任何 MMRM 或 MNAR 敏感性分析（tipping-point、pattern-mixture、delta-adjustment），无法评估结论稳健性。
   - **正确做法**：主分析改 MMRM（按观测值、unstructured 协方差）；并补至少两项敏感性分析（如可控 delta 调整下的 tipping-point analysis、J2R / CIR 多重填补），报告在何种偏移量下结论翻转。

4. [重大] 多重比较未校正，次要终点仅按名义 p 报告
   - **引用**：§2.4「次要终点按名义 p<0.05 报告」；§3.3 腰围 p=0.04、甘油三酯 p=0.04
   - **简析**：两个次要终点均为 p=0.04，恰贴 0.05 线、无任何多重比较控制（Bonferroni / Holm / Hochberg / 层级检验）。在 ≥2 个终点且无 family-wise error 控制下，名义 p=0.04 极易为假阳性；摘要又只挑选这两个「显著」终点呈现，构成选择性报告，无法判断是否还有未报告的非显著次要终点。
   - **正确做法**：预设主/次要终点的多重比较策略（主终点 gatekeeper → 次要终点按层级或 Holm 校正）；按 CONSORT 报告全部预设次要终点（含非显著）。

5. [重大] 子组分析报告了组间差值但无交互检验
   - **引用**：§3.4「BMI≥33 组减重 7.8% 大于 <33 组（5.2%），提示重度肥胖者获益更大」
   - **简析**：子组内点估计的差异（7.8% vs 5.2%）不等于效应修饰的证据——必须检验 treatment × baseline-BMI-stratum 交互项的 p 值。无交互 p，仅凭两个子组点估计大小做「获益更大」的生物学结论是统计错误，且子组分析本身检验功效低，差异很可能是噪声。另子组预设后亦需考虑子组数的多重性。
   - **正确做法**：报告交互项 p 值；只有交互显著才支持效应修饰结论；否则只能描述为「探索性、假设生成」。

6. [一般] ANCOVA 统计前提检验完全缺失
   - **引用**：§2.4 仅声明使用 ANCOVA，未报告任何前提检验
   - **简析**：ANCOVA 要求基线协变量与结局线性、残差正态、方差齐性、各组内回归斜率齐性（homogeneity of slopes）。论文未报告任何一项检查，也未说明是否检验协变量×组别交互（如斜率不齐则 ANCOVA 不适用）。对于体重这类通常近似正态但仍需验证的结局，缺少前提检验影响结论可信度。
   - **正确做法**：报告残差正态性（如 Shapiro-Wilk 或 Q-Q）、Levene 方差齐性、协变量×组别交互项检验；前提不满足时改用稳健方法（如 Welch ANCOVA、bootstrap CI）。

7. [一般] 效应量/离散度不全，且分析人群未明确界定
   - **引用**：§3.2 / §3.3 仅给组间差值与 CI，无各组 SD、无标准化效应量；全文未出现 ITT / mITT / PP 字样
   - **简析**：(a) 缺少各组体重变化的 SD，读者无法换算 Cohen's d / Hedges' g，无法与既往试验（SCALE 等）的效应量对标。(b) 随机化 200 例但实际完成 88+76=164 例，LOCF 分析究竟基于 200（mITT）还是 164（PP）未明说；若用完成者分析则引入选择偏倚，与 LOCF 的表述相互冲突。(c) ANCOVA 未提及是否纳入随机化分层因子作为协变量（标准做法），分层信息缺失。
   - **正确做法**：报告各组均值 ± SD 与标准化效应量；明确分析人群（首选 ITT/mITT 并经 CONSORT 流程图交代排除）；若随机化按 BMI 分层，则 ANCOVA 须纳入该分层因子。

## 总评（拒）
主要终点的 95% CI 与 p 值在数学上无法共存（区间甚至不以点估计为中心），叠加缺失样本量/功效、LOCF 在不对称脱落下的偏倚、次要终点无多重比较校正、子组无交互 p——统计学层面的可信度从地基处崩塌，必须拒稿；即便数据真实，也需重新分析并补齐 power、敏感性、交互与多重比较后重投。
